tpxForms is an enhanced version of some of the html form elements written in LESS and compiled in css.

Created by: @themepixels

Getting Started

tpxForms is currently in initial release (v1.0) and is licensed under MIT. To get started click any of the two download buttons below to see what suits your needs.


Download CSS Compiled and minified CSS
Download Source Includes less files, unminified css and docs


or you can install it using Bower 

$ bower install tpxforms

After downloading the css file, add the code below to your <head> of your page. 

<link rel="stylesheet" href="tpxforms.min.css">

Radios

Radio buttons are used when you want to let the visitor select one option from a set of alternatives.


Use the available syntax below to use the enhanced radio box.

<label class="tpx-radio">
  <input type="radio">
  <span></span>
  Default Radio
</label>

To create a custom design, we use the <span> element as an indicator to the input radio which is hidden within the <label>.

Default Radio

Start with this basic radio by copying the syntax given above.


Inversed Radio

Create an inversed color of the default radio.

<label class="tpx-radio tpx-radio-inverse">...</label>

Radio Sizes

Make larger or smaller radio by adding .tpx-radio-sm, .tpx-radio-xs, or .tpx-radio-lg 

<label class="tpx-radio tpx-radio-lg">...</label>
<label class="tpx-radio tpx-radio-sm">...</label>
<label class="tpx-radio tpx-radio-xs">...</label>

Radio Disabled

Add the disabled attribute to <input> within the label.

<input type="radio" disabled>

Colored Radio

Use any of the available radio classes to create a colored radio button..

<label class="tpx-radio tpx-radio-primary">...</label>
<label class="tpx-radio tpx-radio-primary tpx-radio-inverse">...</label>
<label class="tpx-radio tpx-radio-success">...</label>
<label class="tpx-radio tpx-radio-success tpx-radio-inverse">...</label>
<label class="tpx-radio tpx-radio-warning">...</label>
<label class="tpx-radio tpx-radio-warning tpx-radio-inverse">...</label>
<label class="tpx-radio tpx-radio-danger">...</label>
<label class="tpx-radio tpx-radio-danger tpx-radio-inverse">...</label>

Using LESS

tpxForms is built using LESS. You can take advantage of it by adding more colors or adding even larger radios/checkboxes very easily.

By using LESS, you are required to use Grunt to compile and minified it to css. If you haven't used Grunt before, be sure to check out the Grunt's Getting Started guide, as it explains how to create a Gruntfile as well as install and use Grunt plugins.


Variables

For radio box, there are two important variables the .indicator-radio-size and .radio-dot-size. By changing the value of the two variables it will also change the size of available sizes for the radio automatically, also same way with the checkboxes. 

// Radio Variables
// Note: @indicator-radio-size and @radio-dot-size must be even numbers
@indicator-radio-size:      18px;
@radio-dot-size:            10px;

// Do not touch this unless you know what you're doing. :)
// This will center the dot position automatically
@radio-dot-position:        (@indicator-radio-size - ((@radio-border-width * 2px) + @radio-dot-size)) / 2;

// Checkbox Variables
@indicator-checkbox-size:   18px;

// Indicator border width for radio and checkbox
@radio-border-width:    1px;
@checkbox-border-width: 1px;

// Space between indicator and the text label
@indicator-space:           10px;

Mixins

One of the best thing to do is that you can easily add more colors to radio/checkbox by adding just one line of code in LESS.

// Adding your own color to radio. Ex: .colored-radio(my-pink-radio, #f818a1);
.colored-radio(@selector, @color);

// Ex: .colored-checkbox(my-pink-checkbox, #f818a1);
.colored-checkbox(@selector, @color);

// To use
<label class="tpx-radio my-pink-radio"> ...
